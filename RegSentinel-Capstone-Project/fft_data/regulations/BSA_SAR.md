# Suspicious Activity Report (SAR)

**Citation:** 31 CFR § 1020.320  
**Authority:** FinCEN — Bank Secrecy Act  
**Form:** FinCEN Form 111  

## Summary

Banks must report known or suspected violations and suspicious transactions relevant to possible law violations.

## Key Requirements

- File a SAR for suspicious transactions aggregating $5,000 or more when a suspect can be identified ($25,000 if no suspect).
- File within 30 calendar days of initial detection (60 days if no suspect identified).
- Maintain SAR confidentiality — do not disclose to the subject.
- Retain SAR and supporting documentation for five years.

## Detection Indicators

- Transactions with no apparent business or lawful purpose.
- Activity inconsistent with the customer's known profile.
- Patterns suggesting structuring, layering, or funnel-account behaviour.

## Enforcement

Civil and criminal penalties for failure to file; SAR confidentiality breach is separately penalised.
